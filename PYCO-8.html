Consolidated Python API Hooks
Input & Rendering
engine.get_joy_x() / y()
Retrieves calculated float vector output arrays [-1.0 to 1.0].
engine.is_btn_pressed(button_string)
Queries active multi-touch state inputs for `"A"`, `"B"`, `"X"`, or `"Y"`.
engine.adjust_camera_yaw(radians)
Alters projection angle matrices dynamically.
engine.draw_stacked_sprite(id, x, y, z, rot)
Blits layered voxel slices to render 3D assets with accurate shading.
engine.draw_world_map()
Assembles background terrain matrices directly onto the active game engine frame.
Audio
engine.play_sfx(id, volume=1.0)
Triggers a defined chiptune sound effect. Volume clamped to [0.0, 1.0]. Invalid IDs are silently ignored.
engine.play_music(pattern_id, loop=True)
Starts a music pattern from step 0. Stops any currently playing pattern first.
engine.stop_music()
Stops the music sequencer and resets step counter to 0. Safe to call when nothing is playing.
engine.set_master_volume(vol)
Sets master audio volume [0.0, 1.0]. Values outside range are clamped.
Physics & Collision
engine.register_entity(id, x, y, w, h, solid=True)
Registers an entity with a bounding box for collision. Duplicate IDs overwrite previous registration.
engine.unregister_entity(id)
Removes an entity from the collision spatial index.
engine.move_entity(id, dx, dy)
Moves entity with wall-sliding collision resolution. Returns {x, y, collisions, events}.
engine.check_overlap(id_a, id_b)
Returns true if two registered entity bounding boxes share area. Symmetric and safe for invalid IDs.
engine.register_trigger(id, x, y, w, h, event_type, payload)
Defines a trigger zone that fires events when entities enter. Types: "door", "chest", "zone", "custom".
Gameplay & Screens
engine.transition(type, duration, callback_name)
Plays a screen transition effect. Types: "fade", "wipe_left", "wipe_right", "iris_close", "iris_open".
engine.is_transitioning()
Returns true while a screen transition animation is active.
engine.register_screen(id, enter_fn, update_fn, exit_fn)
Registers a named game screen with lifecycle callbacks.
engine.switch_screen(id, transition_type)
Transitions to a registered screen, invoking exit/enter callbacks.
engine.get_current_screen()
Returns the ID of the currently active screen, or null if none.
engine.create_combat_entity(id, hp, attack, defense)
Registers a combat entity with RPG stats. HP [1, 99999], Attack/Defense [0, 99999].
engine.calculate_damage(attacker_id, defender_id)
Returns attack minus defense, floored at 1. Returns 0 for invalid IDs.
engine.apply_damage(entity_id, amount)
Reduces entity HP by amount, clamping at 0. Returns 0 for invalid IDs.
engine.get_combat_entity(id)
Returns {hp, attack, defense} for a registered combat entity.
engine.generate_dungeon(width, height, seed)
Returns a 2D grid of "wall"/"floor"/"corridor" cells. Deterministic from seed. Dimensions: 5–128.
engine.spawn_entities_in_region(template_id, count, region)
Places entities at floor cells in {x, y, w, h} region. Count: 1–256.
engine.set_timer(id, duration, callback_name, repeat)
Schedules a callback after duration seconds. Repeat=True for recurring. Max 64 timers.
engine.clear_timer(id)
Cancels a scheduled timer. Safe to call with non-existent IDs.
Multiplayer (P2P)
engine.host_session(room_code)
Creates a WebRTC session with a 4–8 character alphanumeric room code. Supports up to 4 peers.
engine.join_session(room_code)
Connects to an active host via room code. Times out after 10 seconds on failure.
engine.disconnect()
Closes all peer connections and notifies connected peers.
engine.broadcast_state(entity_id, state_obj)
Sends entity state to all peers. Payload must be ≤1024 bytes serialized.
engine.send_message(peer_id, type, data)
Sends a direct message to a specific peer. Rate-limited to 30 msg/s per peer.
engine.get_connection_state()
Returns current state: "disconnected", "connecting", "hosting", or "joined".
